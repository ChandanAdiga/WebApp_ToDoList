# WebApp_ToDoList : Backend - NodeJS

This is the backend application of typical ToDo list web app, based on NodeJS framework. 

## Steps:

### 1. Project setup : NodeJS, Express
    NodeJS : V14+

    $mkdir backend
    $cd backend
    $npm init
    $npm install express
    $touch index.js

### 2. Run the backend:
    $npm start
    (OR)
    $node index.js
